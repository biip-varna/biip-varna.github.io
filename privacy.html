Политика за поверителност

Последна актуализация: март 2026 г.

Нашият ангажимент: без проследяване, без бисквитки

Българският институт за международна политика (БИМП) е ангажиран с уважение към поверителността ви. Този уебсайт не:

  • Използва бисквитки от какъвто и да е вид
  • Събира лични данни
  • Проследява индивидуални посетители
  • Използва рекламни мрежи
  • Споделя данни с трети страни
  • Извършва анализи или профилиране на потребители

Статичен уебсайт

Това е статичен уебсайт, хостван в GitHub Pages. Цялото съдържание се предоставя като обикновени HTML, CSS и JavaScript файлове. Не се извършва сървърна обработка на данни за посетители.

Услуги на трети страни

Единствената външна услуга, с която се свързва уебсайтът, е:

  • Open-Meteo (open-meteo.com) — използва се за показване на текущото метеорологично време във Варна на началната страница. Тази заявка се изпраща директно от браузъра ви към API на Open-Meteo и не включва лично идентифицираща информация. Open-Meteo не задава бисквитки и не проследява потребители.

Бутони за споделяне в социалните мрежи

Статиите включват незадължителни бутони за споделяне (LinkedIn, X/Twitter, Facebook, Telegram, WhatsApp, имейл). Това са обикновени хипервръзки — те не зареждат външни скриптове и не задават бисквитки. При кликване ще бъдете пренасочени към съответната платформа, която прилага собствена политика за поверителност.

Хостинг в GitHub Pages

Сайтът е хостван в GitHub Pages (GitHub, Inc. / Microsoft). GitHub може да обработва IP адреси като част от стандартната инфраструктура за уеб хостинг. Вижте декларацията за поверителност на GitHub за подробности.
