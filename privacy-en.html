Privacy Policy

Last updated: March 2026

Our commitment: no tracking, no cookies

The Bulgarian Institute for International Politics (BIIP) is committed to respecting your privacy. This website does not:

  • Use cookies of any kind
  • Collect personal data
  • Track individual visitors
  • Use advertising networks
  • Share data with third parties
  • Run analytics or user profiling

Static website

This is a static website hosted on GitHub Pages. All content is served as plain HTML, CSS, and JavaScript files. No server-side processing of visitor data occurs.

Third-party services

The only external service this website contacts is:

  • Open-Meteo (open-meteo.com) — used to display current weather conditions in Varna on the homepage. This request is made from your browser directly to the Open-Meteo API and does not include any personally identifiable information. Open-Meteo does not set cookies and does not track users. See their terms.

Social sharing links

Articles include optional social sharing buttons (LinkedIn, X/Twitter, Facebook, Telegram, WhatsApp, Email). These are standard hyperlinks — they do not load external scripts and do not set cookies. Clicking them will take you to the respective platform, which will apply its own privacy policy.

GitHub Pages hosting

This site is hosted on GitHub Pages (GitHub, Inc. / Microsoft). GitHub may process IP addresses as part of its standard web hosting infrastructure. See GitHub's Privacy Statement for details.

Contact
